pour lancer le projet : tns run dans le dossier TD4

Première utilisation : s'inscrire ===> un mot de passe s'affichera le noter pour se connecter à l'application 

Se connecter on arrive sur une page vide avec un bouton add pour ajouter un todo 
Si un todo est done la poubelle apparait et on peux le supprimer 
on clique sur un todo on accede au détail de la todo on peut passe la todo en done ou notdone 
on peut update le nom de la todo 
On peut aussi directement changer done en not done ou inverse directement en cliquant sur "done" ou "not done" sur la page principale
Sur le deuxième onglet on peut reset le mot de passe ce qui nous donne un nouveau mot de passe 
On peut se déconnecter aussi 
Si le token est toujours valide lors de l'ouverture de l'application cela nous ammène directement sur la page d'acceuil
